# C++ Core Algorithms & CS Review Library

This project is a personal C++ library focused on re-implementing commonly used algorithms, data structures, and core computer science concepts.  
The goal is interview preparation, deep understanding, and long-term reference.

---

## 1. Data Structures

### Linear

- [] Dynamic Array (vector-like)
- [] Singly Linked List
- [] Doubly Linked List
- [] Stack
- [] Queue
- [] Deque
- [] Circular Buffer

### Hashing

- [] Hash Table (chaining)
- [] Hash Table (open addressing)
- [] Custom hash functions
- [] Load factor & rehashing strategies

### Trees

- [] Binary Tree
- [] Binary Search Tree
- [] AVL Tree
- [] Red-Black Tree
- [] Heap (min/max)
- [] Trie (prefix tree)

### Graphs

- [] Graph representations (adjacency list / matrix)
- [] Weighted vs unweighted graphs
- [] Directed vs undirected graphs
